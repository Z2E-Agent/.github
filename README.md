## What is Z2E?

Z2E (Zero to Exploitation) is an autonomous AI-powered penetration testing framework that streamlines the entire security assessment workflow — from initial reconnaissance to exploitation.

## Documentation

Visit **[z2e.team](https://z2e.team)** for comprehensive documentation, tutorials, and API references.

## Disclaimer

Z2E is designed for **authorized security testing only**. Always obtain proper written authorization before testing any systems. Unauthorized access to computer systems is illegal.
